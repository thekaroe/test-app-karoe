1. คอนเซ็ปต์ของหนังสือ  
    เน้นนำเสนอถึงการเขียนโปรแกรมคอมพิวเตอร์ด้วยภาษา JAVA ตั้งแต่พื้นฐาน ไปจนถึงระดับการใช้งานจริง สามารถศึกษาได้แม้จะไม่มีพื้นฐานมาก่อน นอกจากนี้ยังได้นำเสนอด้วยโปรแกรม NetbeansIDE ซึ่งเป็นเครื่องมือเขียนโปรแกรมยอดนิยม

2. สารบัญ ผมได้แนบมากับเมล์ฉบับนี้ครับ

3. จุดเด่นที่เหนือกว่าเล่มอื่น ๆ ในท้องตลาด
    เด่นกว่าหนังสือเล่มอื่น เพราะว่า มีเนื้อหาในเรื่องของ JAVA GUI ด้วย และยังพาทำ Workshop ซึ่งเล่มอื่น ๆ ในท้องตลาด ไม่มี และนอกจากนี้ยังนำเสนอด้วยภาษาที่เข้าใจง่าย เน้นการอธิบายด้วยภาพ ไม่มีข้อความเยอะเกินไป และไม่มีศัพย์ทางวิชาการ จนทำให้ทำความเข้าใจลำบาก
    และนอกจากนี้ หลายปีที่ผ่านมา เราได้ทำการวางขายบนเว็บ www.javathailand.com ซึ่งทำให้ผู้คนรู้จักเป็นอย่างดี ในวงการคอมพิวเตอร์ และเป็นที่ยอมรับ (จากปริมาณยอดขายที่ผ่านมา ถือว่าดีมาก)

4. ความหนา และราคา
    ความหนาของหนังสืออยู่ที่ประมาณ 300 - 320 หน้า วางจำหน่ายในราคา 280 บาท

5. ตลาดให้ความสนใจมากน้อยแค่ไหน   มีกลุ่มเป้าหมายที่ชัดเจนหรือไม่   
    ตลาดต้องการหนังสือแบบนี้เป็นอย่างมาก เพราะทั้งในท้องตลาดที่วางจำหน่าย ยังไม่มีเนื้อหาเรื่องของ Java Programming ที่สร้างโปรแกรมแบบ Desktop Application ได้จริง
    กลุ่มเป้าหมายคือ
        1. กลุ่มนักเรียน นักศึกษา ครู อาจารย์ ที่อยู่ในแวดวงของคอมพิวเตอร์
        2. คนทำงาน และคนสนใจในด้าน การเขียนโปรแกรมคอมพิวเตอร์
        3. บุคคลทั่วไป ที่สนใจศึกษาด้านของ OOP Programming (ซึ่งภาษา JAVA เป็นหัวใจของ OOP)
